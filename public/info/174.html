自考专科行政管理学试题练习（二）

来源：http://wan.zkcedu.com    作者：安徽自考网   时间：2018-12-26    浏览：
次

一、单项选择题(在每小题的四个备选答案中，选出一个正确答案，并将正确答案的序号填在题干的括号内。每小题1分，共20分)

1. 行政管理的主体是国家( )

A. 权力机关 B. 立法机关

C. 司法机关 D. 执行机关


2. 现代人事制度划时代的标志是( )

A. 任用制度 B. 职位分类

C. 工资制度 D. 考核制度


3. 就职位的法律性质和是否授予人担任，对行政领导者职位的不同情况所做的区别称为( )

A. 职位划分 B. 职位区分

C. 职位分类 D. 职位分别


4. 办公室管理的根本特征是( )

A. 政策性 B. 综合性

C. 辅助性 D. 服务性


5. 现代行政咨询研究中一项重要的方法是( )

A. 经验分析方法 B. 理论分析方法

C. 民意测验方法 D. 头脑风暴方法


6. 以下属于监督社会行为中介组织的是( )

A. 商会 B. 检测中心

C. 研究会 D. 人才交流中心


7. 把国家权力分为立法权、行政权和司法权的西方思想家是( )

A. 孟德斯鸠 B. 卢梭

C. 潘恩 D. 杰斐逊


8. 一个国家在一定时期内实行由若干税种构成的税收制是( )

A. 单一税 B. 实物税

C. 配赋税 D. 复合税


9. 实行行政法治的前提是( )

A.严格执法 B. 加强立法

C. 司法独立 D. 社会认同


10. 有助于协调人与人之间的关系，把人的行为引向社会共同目标的实现的行政方法是( )

A. 思想教育方法 B. 目标管理方法

C. 社会学方法 D. 心理学方法


11. 最基本的行政道德要求是( )

A. 以德照人 B. 以廉勉人

C. 廉洁奉公 D. 勤政为民


12. 标准比较法评定的是特定行政活动的( )

A. 效益 B. 效果

C. 效能 D. 功能


13. 我国第三阶段(1992年10月至1997年8月)行政改革的根本途径是实现( )

A. 转变职能 B. 理顺关系

C. 精兵简政 D. 政企分开


14. 贯穿于行政管理各个方面和全过程的职能是( )

A. 计划职能 B. 组织职能

C. 协调职能 D. 控制职能


15. 构成国家行政体制最基本的要素是( )

A. 公务人员 B. 政府机构

C. 公务制度 D. 人事法规


16. 抽样行政调查中最关键的环节是( )

A. 样本抽选 B. 检查

C. 推算 D. 结论


17. 整个协调活动的中心环节是( )

A. 协调准备阶段 B. 协调实施阶段

C. 协调督办阶段 D. 协调反馈阶段


18. 公平合法行政的前提是( )

A. 行政职责法定化 B. 行政职权法定化

C. 行政责任法定化 D. 行政权限法定化


19. 以卡斯特、罗森茨韦克为代表的现代组织理论是( )

A. 权变组织理论 B. 柔性组织理论

C. 系统分析组织理论 D. 刚性组织理论


20. 财务行政的目的是( )

A. 保持国家财政收支平衡，保证经济正常运行

B. 实现资源优化配置，提高经济运行效率

C. 调解国家、企业和个人之间的分配关系，维持社会公平

D. 提高财物资源的管理效率，更好地满足社会公共需要


二、多项选择题(在每小题的五个备选答案中，选出二至五个正确的答案，并将正确答案的序号分别填在题干的括号内，多选、少选、错选均不得分。每小题1分，共10分)

1. 行政权力的特点是( )

A. 它是公共权力 B. 它具有合法性 C. 它是一种执行权力

D. 它是一种强制性权力 E. 它具有“一元性”


2. 以组织内部行使最高决策的人数划分，行政组织可分为( )

A. 集权制 B. 首长制 C. 分权制

D. 委员会制 E. 混合制


3. 公务员制度的基本原则是( )

A. 依法行政 B. 分类管理 C. 公开平等

D. 竞争择优 E. 智能互补与结构合理原则


4. 我国行政首长负责制的主要内容是( )

A. 行政首长领导本级政法的工作

B. 副行政首长和其他政府成员只是行政首长的助手，协助他工作，对他负责

C. 我国的行政首长负责制是集体负责制

D. 实行人民政府工作会议制度

E. 最常见和最主要的工作会议有各级政府全体会议和各级政府常务会议


5. 机关后勤事务管理改革的方向是( )

A. 市场化 B. 科学化 C. 企业化

D. 社会化 E. 民营化


6. 信息的物质载体有( )

A. 文字 B. 胶片 C. 磁带

D. 声波 E. 电波


7. 财务行政的主体在我国主要是指中央人民政府和地方人民政府的( )

A. 财政机关 B. 税务机关 C. 人民银行

D. 预算机关 E. 国家金库


8. 行政法治的基本内容是( )

A. 行政权力法定化 B. 行政职权法定化 C. 行政机构法定化

D. 行政程序法定化 E. 行政责任法定化


9. 行政效率的要素有( )

A. 效能要素 B. 效益要素 C. 经济要素

D. 效果要素 E. 时间要素


10. 西方国家传统的科层组织制度的特征主要有( )

A. 分工严密 B. 政府垄断 C. 层级健全

D. 权责明确 E. 制度健全


三、填空题(每空1分，共10分)

1. 行政组织的编制管理包括职能管理、机构管理和_______管理三方面。

2. 人事更新机制主要通过两种方式发挥作用：一是人员更新制度，二是_____更新制度。

3. 行政领导者获得权、责的依据是_______。

4. 行政决策活动的起始点是_______。

5. 行政执行的主体是具有_______的行政机关和社会组织。

6. 行政监督的作用一般概括为预防作用、_______作用、补救作用和评价作用。

7. 人治的机制是_______。

8. 道德自我修养是一种内在的自觉，即_______。

9. 从经济角度讲，提高行政效率的主要途径是降低_______。

10. 就改革基本方式看，行政改革有_______和渐进式两种类型。


四、名词解释(每小题4分，共16分)

1. 行政职能

2. 行政决策体制

3. 财务行政

4. 国家公务员


五、简答题(每小题8分，共24分)

1. 简述行政组织的基本要素。

2. 简述责任行政的主要内容。

3. 简述当代西方国家行政改革的基本趋势。


六、论述题(每小题10分，共20分)

1. 联系我国实际分析行政执行的障碍。

2. 试论述加强我国行政监督的途径。
