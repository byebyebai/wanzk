自学考试企业会计学试题练习

一、单项选择题(本大题共20小题，每小题1分，共20分)

1、会计的首要职能是( )

A、会计监督 B、会计记录 C、会计报告 D、会计核算


2、将一个持续经营企业的生活经营活动划分为若干连续、相等的期间，这一会计假设称为( )

A、会计主体 B、会计分期 C、货币计量 D、权责发生制


3、企业日常经营活动的现金支取需开立的银行结算账户是( )

A、一般存款账户 B、专用存款账户 C、临时存款账户 D、基本存款账户


4、某企业购入股票作为交易性金融资产，购入价格11900元，其中包括已选出但尚未发放的现金股利1500元，另支付相关税费100元。则该交易性金融资产的入账价值是( )

A、10400元 B、10500元 C、11900元 D、12000元


5、某公司2015年年初“坏账准备”账户贷方余额为3000元，年末应收账款余额未900000元，估计坏账率为5%，则年末应计提的坏账准备为( )

A、42000元 B、45000元 C、45150元 D、48000元


6、某公司销售商品，价款为20000元，增值税税额为3400元。取得一张面值为23400元、期限4个月、年利率5%的商业汇票。该汇票的到期值为( )

A、20000元 B、23400元 C、23790元 D、24570元


7、我国会计制度规定，应计入存货采购成本的是( )

A、享有的商业折扣 B、享有的现金折扣 C、入库后的仓储费 D、运输途中非合理损耗


8、物价上涨时，会导致企业高估当期利润的发出存货计价方法是( )

A、个别计价法 B、先进先出法 C、加权平均法 D、移动平均法


9、“在途物资”账户的期末借方余额表示( )

A、库存材料的实际成本 B、库存材料的计划成本 C、在途材料的实际成本 D、在途材料的计划成本


10、下列关于固定资产会计处理的说法中，不正确的是( )

A、日常维修支出计入固定资产成本

B、建造固定资产过程中耗用的人工费用计入在建工程成本

C、固定资产达到预定可使用状态后的利息支出计入财务费用

D、固定资产的更新改造支出符合资本化条件的计入固定资产成本


11、下列各项中，属于投资性房地产的是( )

A、自用的房屋 B、出租的房屋 C、租入的房屋 D、出借的房屋


12、下列票据中，通过“应付票据”账户核算的是( )

A、支票 B、商业汇票 C、银行汇票 D、银行本票


13、下列各项中，不属于职工薪酬内容的是( )

A、职工奖金 B、住房公积金 C、职工福利费 D、职工报销的差旅费


14、企业以面值发行债券称为( )

A、折价发行 B、议价发行 C、溢价发行 D、平价发行


15、当月销售商品因质量问题发生销售退回，正确的会计处理是( )

A、增加管理费用 B、冲减营业外收入 C、增加营业外收入 D、冲减主营业务收入


16、大量大批多步骤生产的产品通常适用的产品成本计算方法是( )

A、品种法 B、分批法 C、分步法 D、约当产量法


17、下列各项中，属于期间费用的是( )

A、审计费 B、车间办公费

C、生产工人工资 D、生产设备折旧费


18、下列各项中，属于企业营业收入的是( )

A、接受捐赠收入 B、取得罚款收入 C、提供劳务收入 D、出售无形资产净收益


19、下列各项中，不属于现金流量表中现金范围的是( )

A、库存现金 B、可随时支取的活期存款

C、不能随时支取的定期存款 D、提前通知银行便可支取的定期存款


20、下列指标中，反映企业营运能力的是( )

A、每股盈余 B、速动比率 C、销售净利率 D、存货周转率


二、多项选择题(本大题共5小题，每小题2分，共10分)

21、下列各项中，属于会计计量属性的有( )

A、现值 B、重置成本 C、历史成本 D、公允价值 E、可变现净值


22、投资按其性质，可分为( )

A、对外投资 B、对内投资 C、债权性投资 D、权益性投资 E、混合性投资


23、下列项目中，属于存货的有( )

A、原材料 B、在建工程 C、生产成本 D、工程物资 E、库存商品


24、下列各项税金中，不计入管理费用的有( )

A、印花税 B、增值税 C、消费税 D、企业所得税 E、城市维护建设税


25、根据我国企业会计准则规定，企业财务报表应包括( )

A、利润表 B、报表附注 C、资产负债表 D、现金流量表 E、所有者权益变动表

四、账务处理器(本大题共3小题，每小题10分，共30分)

30、甲公司有关长期股权投资的经济业务如下：

(1)2014年1月1日，甲公司以银行存款3335000元购入A公司20%的股份，另支付相关税费5000元，能够对A公司施加重大影响，采用权益法核算。投资时A公司可辨认净资产公允价值为16700000元。

(2)2014年12月31日，A公司当年实现净利润1100000元。

(3)2015年3月1日，A公司宣告分配现金股利300000元。

(4)2015年4月15日，收到现金股利存入银行。

要求：

(1)编制2014年1月1日甲公司取得长期股权投资的会计分录;

(2)编制2014年12月31日甲公司应享有A公司净利润份额的会计分录;

(3)编制2015年3月1日甲公司享有A公司分配现金股利份额的会计分录;

(4)编制2015年4月15日甲公司收到现金股利的会计分录。

(“长期股权投资”科目要求写明细科目)


31、乙公司为增值税一般纳税人，有关固定资产业务如下：

(1)2015年10月购入一台需要安装的生产设备，去的增值税专用发票，价款600000元，增值税税额102000元，以银行存款支付。

(2)设备安装过程中，发生安装人员工资8000元，尚未支付。2015年12月设备安装完成交付使用。

(3)该设备预计使用10年，预计净残值率5%，采用平均年限法计提折旧。

要求：

(1)编制购入设备的会计分录;

(2)编制安装设备的会计分录;

(3)编制设备交付使用的会计分录;

(4)计算2016年应计提的折旧额;

(5)编制2016年计提折旧的会计分录。

(“应交税费”科目要求写明细科目)


32、丙公司为增值税一般纳税人，2015年12月发生经济业务如下：

(1)赊销商品，价款1000000元，增值税税额170000元，商品成本800000元。

(2)发行股票800000股，每股面值1元，每股发行价5元，收到股款存入银行。(不考虑股票发行费用)

(3)出售交易性金融资产，售价3000000元，账面价值2800000元，其中：成本2800000元，公允价值变动0元。

(4)上月销售商品，因质量问题，经协商给予购货方10%的折让，货款尚未收到，该商品销售时售价200000元，增值税税额34000元。按规定，发生的折让允许抵减当期增值税。

要求：

(1)编制赊销商品并结转成本的会计分录;

(2)编制发行股票的会计分录;

(3)编制出售交易性金融资产的会计分录;

(4)编制销售折让的会计分录。

(“应交税费”和“交易性金融资产”科目要求写明细科目)


五、计算分析题(本大题共2小题，每小题10分，共20分)

33、丁公司研发一项新技术，2015年初进入开发阶段，发生的研究支出如下：材料费用80000元，人工费用200000元，其他相关费用40000元，其中：符合资本化条件的支出为300000元。2015年7月，该项新技术研发成功取得专利权，预计使用10年。

要求：

(1)不符合资本化条件的研发支出应计入利润表的哪个项目?金融为多少?

(2)研发成功后符合资本化条件的支出应计入资产负债表的哪个项目?

(3)计算2015年该专利权的摊销额;

(4)计算2015年年末该专利权在资产负债表中应填列的金额。


34、戊股份有限公司2015年年初所有者权益总额10000000元，其中：股本2000000元，资本公积4000000元，盈余公积1000000元，未分配利润3000000元。2015年利润总额1960000元，支付税收滞纳金140000元，取得国债利息收入100000元。该公司所得税采用应付税款法核算，适用的所得税税率为25%。2015年年末按净利润的10%提取法定盈余公积，宣告分派现金股利600000元，将资本公积转增股本2000000元。

要求：

(1)计算2015年应纳税所得额;

(2)计算2015年应交企业所得税;

(3)计算2015年净利润;

(4)计算2015年年末留存收益;

(5)分析说明资本公积转增股本对所有者权益总额的影响。
