自学考试专科行政管理学试题练习（一）

一、单项选择题(本大题共30小题，每小题1分，共30分)在每小题列出的四个选项中只有一个选项是符合题目要求的，请将正确选项前的字母填在题后的括号内。

1.20世纪30年代，发现了产量增加与尊重人有密切关系，主张把研究的重点集中于人的行为和人际关系上的是( )

A.管理过程 B.社会系统 C.霍桑实验 D.权变理论

2.微观行政环境诸因素对行政活动的影响是( )

A.主导性的 B.间接的 C.最小的 D.最直接的

3.美国学者高斯于1936年出版并提出了行政管理与行政环境之间关系问题的著作是( )

A.《比较公共行政的模式》 B.《公共行政生态学》

C.《政府生态学》 D.《美国社会与公共行政》

4.里格斯所论述的融合型行政模式的特征主要是( )

A.经济基础是自由经济或管制经济 B.行政风范体现平等主义

C.政府的制度、法规不健全 D.官僚的职位重于行政政策

5.下列各项中既包含政府的行政职责，也包含政府的行政功能的是( )

A.行政计划 B.行政领导 C.行政职能 D.机关管理

6.当代西方资本主义国家政府职能的重点在于( )

A.政治 B.民主 C.文化 D.经济

7.提出双模式组织标志的三种类型的压力等理论的西方现代管理学家是( )

A.品乔特兄弟 B.哈耶克 C.巴拉密 D.德鲁克

8.下列各项中属于西方国家公务员制度基本特征的是( )

A.政务官实行常任制

B.政务官与内阁共进退

C.事务官实行常任制，政务官与政党共进退

D.事务官与政党共进退

9.我国国务院通过的《国家公务员暂行条例》开始施行于( )

A.1987年10月1日 B.1990年10月1日

C.1992年10月1日 D.1993年10月1日

10.奠定了美国公务员制度基础的是( )

A.彭德尔顿法 B.绅士政府

C.政党分肥制 D.恩赐官职制

11.互通情报，共商重大问题的日常的具体行政领导制度是( )

A.信访制度 B.通报征询制度

C.民主生活会制度 D.办公会议制度

12.行政工艺最主要、最基本的内容可概括为( )

A.研究提高行政质量的技术手段和技术条件

B.研究行政领导者在领导活动中所采取的各种手段、方法

C.研究保障机关正常运转所需的工作和生活条件

D.研究行政机关实现管理目标所采取的方式、手段和技巧

13.各省、自治区、直辖市政府本级总预算的汇总预算称为( )

A.中央预算 B.地方预算

C.部门预算 D.单位预算

14.下列税法中属于税收程序法的是( )

A.《中华人民共和国营业税条例》 B.《中华人民共和国税收征收管理法》

C.《中华人民共和国增值税条例》 D.《中华人民共和国消费税暂行条例》

15.决策时环境条件的发生不能测定、不能控制，因而决策后果不能加以确定，具有较大风险的决策，被称为( )

A.风险型决策 B.随机型决策

C.统计型决策 D.不确定型决策

16.在引起差异的原因尚未查清又急需处理时，采取暂时性措施抑制问题发展的方案，被称为( )

A.积极方案 B.消极方案

C.临时方案 D.应变方案

17.行政咨询必须遵循的基本原则是( )

A.现实可操作性原则 B.理论分析原则

C.民意测验原则 D.民主化原则

18.咨询成果质量与咨询人员素质之间( )

A.成正比关系 B.成反比关系

C.不构成比例关系 D.没有必然联系

19.决定着决策方案能否实现及实现的程度在于( )

A.决策内容是否符合实际 B.决策方案是否优化

C.行政执行 D.决策是否代表人民利益

20.政府系统内部有隶属关系和上下级指导关系的政府机构、部门或成员之间的协调，被称为( )

A.政府与国家权力机关的协调 B.政府与国家司法机关的协调

C.政府内部的纵向协调 D.政府内部的横向协调

21.下列各项中属于行政信息系统运行保障的是( )

A.信息传输手段 B.信息工作制度

C.信息工作机构 D.信息工作人员

22.我国唐代设置的独立的监察机关是( )

A.中书省 B.门下省 C.督察院 D.御史台

23.行政系统内部设立的专门监督机构实施的行政监督是指( )

A.职能监督 B.行政监察 C.主管监督 D.审计监督

24.我国国务院制定《行政法规制定程序暂行条例》的时间是( )

A.1986年 B.1987年 C.1990年 D.1992年

25.在1988年、1993年政府机构改革中，国务院机构的职能配置、内设机构、人员编制是以( )

A.自定方案形式经国务院批准后付诸实施的

B.国务院决议形式付诸实施的

C.国务院通知形式付诸实施的

D.组织立法形式付诸实施的

26.衡量一个国家民主、文明程度的重要标志是( )

A.公平与正义 B.公开与无私

C.廉洁与严明 D.独立与刚正

27.经济方法的根本特征是( )

A.间接性 B.有偿性 C.利益性 D.多样性

28.在下列各项标准中，不属于中间管理层工作质量标准的是( )

A.对上级命令执行的程度 B.服务态度好坏

C.能否及时有效地处理突发事件 D.管理系统内部协调一致的程度

29.党风、政风、社会风气及公民整体效率意识是影响行政效率的( )

A.环境因素 B.组织因素 C.人员因素 D.科学技术因素

30.下列各项中属于日本政府于1984年修改的法律是( )

A.国家行政组织法 B.国家公务员法

C.国家宪法 D.国家编制法


二、多项选择题(每小题1分，共17分)在每小题列出的五个选项中有二至五个选项是符合题目要求的，请将正确选项前的字母填在题后的括号内。多选、少选、错选均无分。

31.行政管理学内容丰富，下列各项中属于“硬件”部分的是( )

A.行政决策 B.行政组织 C.行政沟通

D.行政机构 E.行政执行

32.学习与研究行政管理学的方法有( )

A.理论联系实际的方法 B.案例分析方法 C.历史研究方法

D.比较研究方法 E.调查研究方法

33.社会环境中的政治环境包括( )

A.一国中占统治地位的阶级及其政党状况

B.社会各阶级在国家中的地位

C.政体

D.国家结构形式

E.立法、执法、守法和监督的制度

34.下列各项中属于人事行政特殊职能的是( )

A.公职人员的录用 B.公职人员的开发 C.公职人员的维持

D.公职人员的使用 E.公职人员的计划

35.下列各项制度中属于公务员制度管理环节的是( )

A.考核制度 B.奖励制度 C.纪律制度

D.职务升降制度 E.培训制度

36.根据行政领导权力的控制和运用程度的不同，可以把领导方式分为( )

A.强制方式 B.集权方式 C.说服方式

D.分权方式 E.均权方式

37.运用授权艺术应坚持的原则有( )

A.授权留责 B.适度合理授权 C.因人择事、视事授权

D.逐级授权 E.加强监督控制，防止反向授权

38.我国机关后勤服务机制要实现如下转变( )

A.企业型向事业型转变 B.事业型向企业型转变 C.经营型向福利型转变

D.福利型向经营型转变 E.依赖型向开拓型转变

39.行政执行对行政决策的作用主要有( )

A.决定着决策方案能否实现 B.决定着决策方案的实现程度

C.是检验决策是否正确的依据 D.是使原决策得到修正和完善的根本途径

E.是形成新的行政决策的重要依据

40.制定行政执行计划应遵循的原则有( )

A.民主集中原则 B.量力而行原则 C.统筹原则

D.重点原则 E.稳定性原则

41.对组织系统关系的行政协调方法主要有( )

A.坦诚引导法 B.目标导向法 C.利益平衡法

D.沟通协商法 E.奖惩并用法

42.我国行政系统外部监督的主体有( )

A.司法机关 B.社会团体 C.国家权力机关

D.执政党 E.公民

43.行政监督的重要作用有( )

A.预防作用 B.保护作用 C.改进作用

D.补救作用 E.评价作用

44.行政系统内部的监督主要有( )

A.一般监督 B.行政监察 C.审计监督

D.司法监督 E.方法监督

45.社会主义法制的基本原则主要包括( )

A.民主 B.法律面前人人平等 C.忠实于宪法和法律

D.忠实于人民利益 E.忠实于事实真相

46.行政程序的标准化包括( )

A.形式的标准化 B.领导的标准化 C.程式的标准化

D.实质的标准化 E.控制的标准化

47.间接测定行政效率主要是通过对行政机关效能和效益的评定来估量行政效率的高低。其具体的评定方法有( )

A.行政职能测评法 B.行政功能测评法 C.行政要素评分法

D.标准比较法 E.预期效率比较法

六、案例分析题(8分)

这个案例对我们有哪些启示?
