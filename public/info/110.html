安徽自学考试医学基础模拟试题


一、单项选择题(本大题共25小题，每小题2分，共50分)

1.过敏性休克多由______________引起。( )

A.Ⅰ型超敏反应B.Ⅱ型超敏反应

C.Ⅲ型超敏反应D.Ⅳ型超敏反应


2.弹性动脉是指( )

A.大动脉B.中动脉C.小动脉D.主动脉


3.某患者，便血，血色鲜红并黏附于粪便表面，提示( )

A.痔疮B.上消化道出血

C.坏死性肠炎D.急性菌痢


4.大肠长约( )

A.1米B.1.5米

C.2米D.3米


5.周围神经系统中______________主要分布于内脏平滑肌、心血管和腺体。( )

A.脑神经B.脊神经

C.躯体神经D.自主神经


6.以下传染病传播途径错误的有( )

A.梅毒胎传传染B.疟疾按蚊叮咬传播

C.流脑消化道传播D.艾滋病输血传播


7.某男，65岁，患高血压病多年，坚持服药，现血压保持在125/84mmHg，根据1999年WHO诊断标准，此血压为( )

A.理想血压B.正常血压

C.临界高血压D.高血压Ⅰ期


8.正常脑脊液呈( )

A.浅红色B.鲜红色

C.无色透明D.浅黄色透明


9.血浆晶体渗透压降低时，红细胞会( )

A.皱缩B.膨胀

C.不变D.先皱缩后膨胀


10.黑热病是由______________引起。( )

A.溶组织内阿米巴B.阴道滴虫

C.血吸虫D.杜氏利什曼原虫


11.______________是血脑屏障的主要结构。( )

A.内皮细胞B.基膜

C.星形胶质细胞D.周细胞


12.肾脏微细结构中______________内含有分泌颗粒，能分泌肾素。( )

A.近球细胞B.致密斑

C.间质细胞D.集合管


13.干性咳嗽常见于( )

A.急性支气管炎初期B.肺炎

C.肺脓肿D.支气管扩张


14.许多药物的代谢和解毒都在( )

A.小肠B.血液

C.肝脏D.脾脏


15.______________是心室肌细胞动作电位区别于神经细胞动作电位的主要特征。( )

A.0期B.1期

C.2期D.3期


16.心脏中下列哪个腔的肌肉最厚?

A.右心房 B.右心室

C.左心房 D.左心室


17.在尿的浓缩、稀释过程中起重要作用的是

A.肾小管 B.肾动脉

C.肾小体 D.肾静脉


18.下呼吸道包括

A.鼻、咽、喉 B.鼻、咽、气管和支气管

C.气管、主支气管及其在肺内的分支 D.气管、主支气管


19.表皮由浅向深依次为

A颗粒层、透明层、生发层、角质层 B.颗粒层、角质层、透明层、生发层

C.角质层、颗粒层、生发层、透明层 D.角质层、透明层、颗粒层、生发层


20.细胞周期中DNA的合成期是

A.S期 B.G1期

C.G2期 D.M期


21.以下不属于传染病的有

A.梅毒 B.高血压

C.疟疾 D.淋病


22.与肺循环无关的部分是

A.肺静脉 B.右心房

C.右心室 D.肺动脉


23.胰岛素属于

A.含氮激素 B.类固醇激素

C.消化液 D.可以口服激素


24.下列不属于呼吸系统的是

A.咽 B.食管

C.气管 D.鼻


25.发作性前胸压榨性疼痛感觉，经休息或含硝酸甘油可缓解的是

A.心肌梗塞 B.冠状动脉粥样硬化

C.心功能不全 D.心绞痛


二、填空题(本大题共5小题，每空1分，共10分)

26.细胞基本结构包括______、______、细胞核三部分。

27.胃可以分为四部分，包括______、______、胃体部、______。

28.脑位于______________内，可分为端脑、_____________、_____________、_____________四部分。

29.肾小管由近侧向远侧依次可分为近曲小管、______________、远曲小管三部分。

30.心射血和______________是形成血压的基本条件。


三、名词解释(本大题共5小题，每小题2分，共10分)

31.心脏前负荷

32.有效不应期

33.激素

34.黄疸

35.超敏反应

36.功能性便秘发生的原因是什么?

39.引起急性腹痛的原因有哪些?
